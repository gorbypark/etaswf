ETA movement flash-based assembly/disassembly archive

ETA, the Swiss watch movement company, provides a wonderful resource for repairing and maintaining some of their movements. It is an interactive flash program that provides step by step dissassembly and assembly instructions. Unfortunately it was built using Adobe Flash, which has been discontinued by Adobe along with all major browser vendors. In order for these wonderful resources to not be lost to time, I have made them available here. It uses ruffle, which is a flash emulator written in the Rust programming language, which is then compiled to web assembly, so it can run in the browser.

6497 calibre (6497-1 & 6497-2)

Errata (6497-1 & 6497-2)

  • There is a (potential) bug in the ruffle flash player that prevents the fonts of the menu to not display. It appears that all other fonts are unaffected. When viewing the 6497 movement, there will be three blank grey buttons. Those grey buttons should say (for english):
    • Dissassembling
    • Assembling
    • Tools
    Click on the required button and everything should work as normal
